## API Endpoints

| Method | Endpoint | Description | Status |
|--------|----------|-------------|--------|
| POST | /api/v1/orders | Create new order | 201 Created |
| GET | /api/v1/orders | Get all orders | 200 OK |
| GET | /api/v1/orders/{id} | Get order by ID | 200 OK |

## Database Schema

**orders table:**
- order_id (UUID, PK)
- order_number (VARCHAR, UNIQUE)
- customer_name (VARCHAR)

- total_amount (DECIMAL)

- updated_at (TIMESTAMP)

- item_id (UUID, PK)

- product_id (UUID)
- product_name (VARCHAR)
- quantity (INTEGER)
- price (DECIMAL)

## How to Run

### Prerequisites
- Java 21
- Maven 3.9+
- Docker Desktop
- PostgreSQL 15 (Docker)

### Start Database
```bash
docker run --name postgres-orderdb \
  -e POSTGRES_PASSWORD=postgres123 \
  -e POSTGRES_DB=orderdb \
  -p 5432:5432 -d postgres:15
```

### Run Application
```bash
cd services/order-service
mvn spring-boot:run
```

### Run Tests
```bash
mvn test
```

Application runs on: http://localhost:8080
